# Typing Speed Test

A simple **Typing Speed Test** application built with **Python** and the **curses** library. This terminal-based project helps users practice typing while measuring their typing speed in **Words Per Minute (WPM)**.

---

## Features

- 🎯 Random typing passages
- ⚡ Real-time WPM (Words Per Minute) calculation
- ✅ Correct characters displayed in green
- ❌ Incorrect characters displayed in red
- ⌫ Backspace support
- 🔄 Restart after completing the test
- ❌ Press **Esc** anytime to exit

---

## Technologies Used

- Python 3
- curses (Terminal UI)
- time module
- random module
